# HTML 드래그 앤 드롭 API

## 드래그 이벤트

HTML 드래그 앤 드롭은 DOM event model과 drag events를 mouse events로부터 상속받는다.

- draggable 요소를 마우스로 선택하고, 마우스 포인터를 droppable 요소로 가져가 마우스 버튼을 때는 것으로 이루어짐.
- 드래그 하는 도중 많은 이벤트가 발생하고 몇몇 이벤트는 여러번 발생하기도 함.
